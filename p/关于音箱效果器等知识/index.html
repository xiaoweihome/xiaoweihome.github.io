音箱前级 / 后级 / 喇叭

前级的作用（AMP）

  • 放大吉他信号
  • 均衡器作用
  • 一般有清音和失真两个通道

后级的作用

  • 前级放大后，还不足以推动喇叭，所以后级是把前级修饰放大后的信号再放大到足以驱动喇叭。

喇叭（CAB）

  • 把前级修饰、后级放大后的信号驱动播放出来。

音箱中的 Effects Loop 功能

Loop 功能的目的就是为了串接效果器。

Effects Loop有两个接口：Send端口是前级的输出端，而 Return 则是后级的输入端

如果把周边效果器串在失真效果器的前面，周边效果会被失真吃掉，得到的声音会很糊；

  • 吉他 — 周边（Chorus/Delay/Reverb） — 失真 — 音箱前级 — 后级功放 — 喇叭

如果把音量踏板放在失真效果器的前面，那么音量踏板就会影响到其失真度，当音量踏板减小，失真度就会降低。

  • 吉他 — 音量踏板 — 失真 — 音箱前级 — 后级功放 — 喇叭

因此，一般都把周边效果器或者踏板串在失真效果器的后面。

其实，失真效果器就相当于音箱的前级，在音箱前级后面串接一些周边效果器之后再输出给音箱的后级功放，以增加音色的可调性范围。

  • 吉他 — 失真 — 音箱前级 — (Send) — 周边 — 踏板 — (Return) — 音箱后级 — 喇叭

Loop端口的一些其他的用途

  1. 将前级输出Send端口直接接入声卡，即可在录音的时候获得电子管音箱前级的音色。

  2. 将合成效果器的输出直接接入音箱的后级输入端口 Return ，绕开绕开音箱的前级，把音箱当纯后级使用。

1
 Fender Twin Reverb Combo · 85W 2×12" 或 2×15"
极致干净、宽广且带有晶莹延伸的混响与合唱感。

Twin Reverb 的前级提供非常干净的增益结构，低频稳定，中频透明，高频延伸但不刺耳。其弹簧混响与 Vibrato/Chorus 创造出空间感，适合乡村、爵士与清音摇滚。对踏板驱动反应优秀。

 Fender Deluxe Reverb Combo · 22W 1×12"
温暖圆润，轻推即可进入管机饱和。

Deluxe Reverb 在较低音量下就能展现管机的自然饱和与谐波，中频略微凸出，使吉他在混音中清晰可辨，常被用于布鲁斯与录音现场。
3
 Vox AC30 Combo · 30W 2×12"（Celestion 常见）
明亮、有“chime”铃声感，高频丰富。

 Vox AC15 Combo · 15W 1×12"

 Marshall 1959 “Plexi” Head · 100W 通常搭配 4×12"

 Marshall JCM800 (2203) Head · 100W 常搭配 4×12"
